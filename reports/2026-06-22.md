## Top 3 du jour

### 1. CVE-2026-45657 : faille wormable dans le noyau Windows (score 9/10)

Microsoft a corrige lors du Patch Tuesday de juin une vulnerabilite critique (CVSS 9.8) dans le noyau Windows. Un use-after-free dans la gestion TCP/IP permet l'execution de code a distance avec privileges SYSTEM, sans authentification ni interaction utilisateur. Le Zero Day Initiative confirme le caractere wormable de cette faille, comparable a EternalBlue (WannaCry). Toutes les versions de Windows 11 (23H2 a 26H1) et Windows Server 2022/2025 sont concernees.

**Pourquoi c'est important :** Toute faille wormable est une urgence absolue en gestion des risques. Les organisations soumises a NIS2 et DORA doivent prioriser le deploiement des correctifs (KB5094126, KB5094125, KB5094128). L'absence de patch sur les systemes exposes pourrait constituer un manquement aux obligations de resilience operationnelle.

### 2. LiteLLM CVE-2026-42271 : les passerelles IA deviennent vecteur d'attaque (score 9/10)

CISA a ajoute a son catalogue KEV la faille CVE-2026-42271 (CVSS 8.7) affectant LiteLLM, une passerelle open-source utilisee pour router le trafic API vers OpenAI, Anthropic, Azure et d'autres fournisseurs LLM. Chainee avec un bypass d'authentification Starlette (CVE-2026-48710), la combinaison atteint un CVSS de 10.0 et permet l'execution de code a distance non authentifiee. La deadline CISA pour les agences federales US est fixee a aujourd'hui, 22 juin 2026.

**Pourquoi c'est important :** Cette vulnerabilite illustre un risque emergent majeur : l'infrastructure IA (gateways, proxies) devient un plan de controle critique mal securise. Un attaquant peut exfiltrer toutes les cles API des fournisseurs LLM, pivoter lateralement et compromettre l'ensemble de la chaine IA. Pour les organisations deploying des agents IA, l'audit de la supply chain IA devient aussi critique que celui des dependances logicielles classiques.

### 3. Fable 5 d'Anthropic suspendu par la Maison Blanche : quand l'IA rencontre le controle des exportations (score 9/10)

L'administration Trump a ordonne la suspension de l'acces aux modeles Fable 5 et Mythos 5 d'Anthropic pour tout ressortissant etranger, invoquant la securite nationale. Le declencheur : un jailbreak decouvert par Amazon ou trois mots ("Fix this code") suffisaient a contourner les garde-fous du modele pour produire du code offensif. Anthropic conteste la proportionnalite de la mesure, arguant qu'un jailbreak ne se "patche" pas comme une vulnerabilite logicielle classique.

**Pourquoi c'est important :** C'est un precedent majeur a l'intersection IA, securite et regulation. Cette decision prefigure la facon dont l'AI Act europeen pourrait traiter les modeles a haut risque. Pour les professionnels GRC, cela souleve la question de la responsabilite des editeurs d'IA face aux usages detournes et de l'applicabilite des controles export aux modeles de fondation.
